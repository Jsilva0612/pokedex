# Configuración de Cuenta en la Nube - PokeDex

Este documento explica el paso a paso realizado para configurar la infraestructura en la nube necesaria para el proyecto PokeDex.

## 1. Creación de la cuenta en Azure
Para este proyecto se utilizó **Microsoft Azure**. Los pasos para obtener la cuenta fueron:
1. Acceder al portal de Azure.
2. Iniciar sesión con el correo institucional de la universidad.
3. Completar la verificación de identidad mediante el código enviado al teléfono móvil.
4. Una vez validado, se activó el crédito de **$100 USD** para uso académico.
